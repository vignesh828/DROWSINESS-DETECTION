# AI Driver Drowsiness Detection System 🚗💤

A real-time safety monitoring system developed using Computer Vision and Deep Learning. This project tracks facial landmarks and triggers an auditory alert if signs of drowsiness are detected.

## Features
- **Real-time Detection:** Uses 68-point facial landmarks for precision.
- **EAR Logic:** Implements Eye Aspect Ratio to detect eye closure.
- **Cross-Platform:** Available as a Web App (Streamlit) and Windows Executable (.exe).
- **Industrial Impact:** Designed for automobile safety and fatigue management.

## Tech Stack
- **Languages:** Python
- **Libraries:** OpenCV, dlib, imutils, winsound
- **Deployment:** PyInstaller, Streamlit
